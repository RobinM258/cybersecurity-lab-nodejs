# cybersecurity-lab-nodejs


Ce projet à pour but d'apprendre à faire un site internet (NodeJs) avec des bases de données (Mariadb)


Installation:

Il faut ajouter un fichier .env avec


DB_HOST=localhost
DB_USER=ton usernane
DB_PASS=ton mot de passe
DB_NAME=cyber_test


Le site permet de faire:
- Une requête en envoyant un user et un email
